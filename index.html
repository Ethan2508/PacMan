1. Déplacez Pac-Man avec les flèches du clavier.
  2. Mangez tous les points blancs pour gagner.
  3. Évitez les fantômes qui se déplacent dans le labyrinthe.
  4. Si un fantôme touche Pac-Man, la partie est terminée.
  5. Appuyez sur le bouton "Recommencez" pour recommencer une partie.
